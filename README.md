# Chernobyl Tycoon (PWA)

Запуск локально:
- Открой `index.html` в static-сервере (или Replit/Netlify).
- Для полной PWA-работы нужен HTTPS (или localhost).

Сборка в APK:
- Пуш в GitHub и используй твой сервис для импорта репо.
- Или: использовать Capacitor / PWABuilder / Bubblewrap для упаковки в Android.
